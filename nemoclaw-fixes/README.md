# NemoClaw Fixes

A collection of troubleshooting guides and fixes for NemoClaw local setup issues. Each file is a standalone post ready to share on the subreddit.

## How to use

- Each `.md` file is a fix/guide written in Reddit-friendly format
- Run `/write-fix-post` in Claude Code after fixing an issue to auto-generate a post
- File naming: `fix-short-description.md` (e.g., `fix-k3d-cluster-start-failure.md`)

## Categories

- **Docker** — Docker install, startup, permissions issues
- **k3d** — Cluster creation, networking, volume issues
- **OpenShell** — Sidecar container, wrapper script, SSH tunneling
- **Inference Keys** — API key encryption, key rotation, provider issues
- **Intel Mac** — Issues specific to x86_64 Macs (openshell sidecar, onboarding time)
- **Apple Silicon** — Issues specific to arm64 Macs
- **Windows/WSL** — WSL2 setup, Docker Desktop integration
